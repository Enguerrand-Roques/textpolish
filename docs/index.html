Why TextPolish

🔒

Fully private

Everything runs on your machine. No API key, no telemetry, no data ever leaves your computer.

⚡

Instant

Casual mode corrects in ~2 s once the model is warm. Pro mode rewrites a full email in ~5 s.

🖥

System-level

Works in any app — VS Code, Notion, Mail, Slack. The panel floats above fullscreen apps.

🔁

Paste back automatically

The corrected text is pasted directly into the source app. No copy-paste steps.

Three modes

PRO

Professional

Full rewrite for emails, reports, LinkedIn. Fixes grammar, improves clarity, keeps your meaning.

CASUAL

Casual

Light touch for SMS, WhatsApp, DMs. Fixes typos and accents — keeps your tone intact.

CUSTOM

Custom

Type any instruction: "Make it punchier", "Simplify for a 10-year-old", "Translate to Spanish".

Model selection — measured, not guessed

Models were evaluated on 10 representative cases (French + English, pro + casual) with Claude Haiku as an automated quality judge. Two rounds of benchmarks informed the final configuration.

Mode Model Median latency Quality score

Installation

macOS (one command)

git clone https://github.com/Enguerrand-Roques/textpolish.git
cd textpolish
bash setup.sh

Requires macOS 13+, Python 3.13, Ollama.

Windows

git clone https://github.com/Enguerrand-Roques/textpolish.git
cd textpolish
pip install -r requirements-windows.txt
python main.py

Requires Python 3.11+, Ollama.

Then pull the models (one-time, ~2 GB): 

ollama pull gemma3:1b   # casual mode
ollama pull gemma3:4b   # professional mode
